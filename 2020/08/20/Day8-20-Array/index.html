Array

  1. Array 数组，可以想象成一串羊肉串，将一个个数给它串起来、存起来。
  2. Java语言中的数组是一种引用数据类型。不属于基本数据类型。数组的父类是Object。
  3. 宿主实际上是一个容器，可以同时容纳多个元素。（数组是一个数据的集合）
  4. 数组在栈中，数组保存的数据在堆内存当中。在栈中的数组保存了在堆中的数据的内存地址，数组通过引用获取数组中的数据。
